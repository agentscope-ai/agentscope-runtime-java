# Contributing to Cookbook

Thank you for your interest in the AgentScope Runtime Java Cookbook!

The AgentScope Runtime Java Cookbook provides practical examples, tutorials, and best practices for building agent applications using AgentScope Runtime Java. We have a friendly community of developers, researchers, and practitioners who are passionate about helping new contributors. We welcome all types of contributions, from adding new tutorials and examples to improving existing documentation and fixing issues.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Git**
- Basic knowledge of **Markdown**

## Installation

Clone the repository and install necessary development dependencies:

```bash
git clone https://github.com/agentscope-ai/agentscope-runtime-java.git
cd agentscope-runtime-java
```

## Managing Documentation

- **Adding Documentation**: Create new Markdown (`.md`) files in the appropriate directory.
- **Deleting Documentation**: Delete the relevant files.

## (Optional) Manual Build Steps

If you prefer to build manually, follow these steps:

1. **Install related dependencies:**

```bash
npm install
```

2. **Start the React application:**

```bash
npm start
```
